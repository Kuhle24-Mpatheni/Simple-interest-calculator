# Simple Interest Calculator

## Project Overview
A Bash script that calculates simple interest based on user input for principal amount, rate of interest, and time period.

## Calculator Details
This calculator implements the standard simple interest formula:
**I = (P × R × T) / 100**
where:
- **P** = Principal amount
- **R** = Rate of interest (percentage)
- **T** = Time period (years)

## Features
- Interactive command-line interface
- Real-time calculation
- Decimal precision support
- Clear input/output display
- Error handling for invalid inputs
